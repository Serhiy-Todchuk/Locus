# S4.R — Memory Bank / Session Scratchpad

## Problem

Agent forgets within a session — learns something useful at turn 3, re-learns it at turn 15. Cline's memory bank, Claude Code's todo list, OpenCode's AGENTS.md all address this.

## Tasks

- [ ] `session_notes` tool: `append(note)`, `read_all()`, `clear()` — persisted to session JSON
- [ ] Auto-inject notes into system prompt (after LOCUS.md) when non-empty
- [ ] UI: activity panel shows notes as a distinct event kind
- [ ] Token-budget guard: warn if notes > 1000 tokens
- [ ] Scope: within-session only (cross-session memory is LOCUS.md's job — don't overlap)
