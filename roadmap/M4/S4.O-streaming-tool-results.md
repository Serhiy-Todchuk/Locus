# S4.O — Streaming / Partial Tool Results

## Problem

`run_command` dumps all output at once after completion. 30 s cmake build = 30 s of silence. LLM gets no intermediate signal. User gets no live feedback.

## Tasks

- [ ] `ITool` extension: optional `execute_streaming(call, ws, on_partial_output)` variant
- [ ] `RunCommandTool` upgraded: emit stdout/stderr chunks via callback as they arrive
- [ ] `IFrontend::on_tool_partial_output(call_id, chunk)` event
- [ ] UI: tool approval panel shows live output area (scrolling, auto-truncate)
- [ ] Truncation policy: if total output > N tokens, keep head + tail; note truncation in LLM-facing result
- [ ] Catch2 tests: chunk ordering, interleaved stdout/stderr
