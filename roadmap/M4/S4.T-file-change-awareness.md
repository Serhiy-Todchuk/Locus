# S4.T — File-Change Awareness Between Turns

## Problem

If user edits a file between LLM turns, agent has no idea. Index updates silently; agent's mental model is stale next turn.

## Tasks

- [ ] Track file mtimes between turns; compute "files changed since last assistant turn"
- [ ] Auto-inject as ephemeral system note on next user turn: `[Files changed since last turn: foo.cpp, bar.h]`
- [ ] Configurable: `agent.notify_external_changes` (default on)
- [ ] Exclude files the agent itself changed (don't echo its own edits back)
