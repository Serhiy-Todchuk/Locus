# S3.C — `IWorkspaceServices` interface

## Problem

[src/tool.h](../../src/tool.h):70 — `WorkspaceContext` is a raw struct with 4 nullable pointers today (`root`, `index`, `embedder`, `workspace`). By end of M4 it needs, at minimum: `CheckpointStore*`, `LspClient*`, `ProcessRegistry*`, `McpRegistry*`, `GitClient*`, `VerifyRunner*`, `SubagentPool*`, `SessionNotes*`, `ActivityLog*`. That's ~12 nullable raw pointers every tool has to null-check — a classic god-struct anti-pattern, and every new subsystem triggers a struct edit + every tool that uses it needs updating.
