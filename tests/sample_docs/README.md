# Sample Documents for Extractor Tests

Real-world sample files used by the document extractor tests
(`PdfiumExtractor`, `DocxExtractor`, `XlsxExtractor` -- see S2.3) and the
S6.2 ZIM reader tests (`tests/test_zim_archive.cpp`).

| File | Source | License |
|---|---|---|
| `sample.pdf` | [IRS Form W-9](https://www.irs.gov/pub/irs-pdf/fw9.pdf) | Public domain (U.S. federal government work, 17 U.S.C. section 105) |
| `sample.docx` | [Apache Tika `testWORD.docx`](https://github.com/apache/tika/blob/main/tika-parsers/tika-parsers-standard/tika-parsers-standard-modules/tika-parser-microsoft-module/src/test/resources/test-documents/testWORD.docx) | Apache License 2.0 |
| `sample.xlsx` | [Apache POI `test-data/spreadsheet/SampleSS.xlsx`](https://github.com/apache/poi/blob/trunk/test-data/spreadsheet/SampleSS.xlsx) | Apache License 2.0 |
| `sample.zim` | [openZIM zim-testing-suite `data/nons/small.zim`](https://github.com/openzim/zim-testing-suite/blob/main/data/nons/small.zim) | Synthetic test fixture, programmatically generated by the openZIM project (no real-world copyrighted content). 40 KB, ZIM v6.1 (new-namespace scheme). |

Do not modify these files -- tests assert on their exact contents
(page counts, heading text, cell values; for the ZIM, the article count and
content round-trip).

The S6.2 ZIM tests resolve their corpus in priority order: `$LOCUS_TEST_ZIM`
override, then this committed `sample.zim` (the default -- keeps the tests
hermetic), then any large real `.zim` under `D:/Projects/WikiZimTest` on the
dev box for a richer-content smoke.
